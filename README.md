# SupplyShock Arena

SupplyShock Arena is an interactive simulation platform designed to explore supply chain decision-making under dynamic and high-pressure conditions.

Users can control key operational variables such as demand, fuel cost, supplier reliability, and resource capacity to observe real-time impacts on system performance.

The system introduces disruption events and allows users to compare different strategies, helping identify the most effective decisions through performance scoring and analysis.

## Key Features

- Interactive control panel for system inputs  
- Dynamic disruption events (fuel spikes, supplier failure, demand surge)  
- Real-time KPI tracking (Delay Risk, Cost Impact, Throughput, Score)  
- Scenario preview before saving  
- First vs Second scenario comparison  
- Winner logic with explanation  
- Downloadable scenario data  

---

## Objective

The goal is to maintain system stability under pressure by:

- Keeping Delay Risk below 40%  
- Maintaining Demand above 60  
- Achieving a Score above 70
